# Hand Gesture Desktop Controller

Control your computer using hand gestures detected through your webcam.

## Features
- Cursor control with hand
- Left click (thumb + middle finger or index finger)
- Right click (thumb + ring finger)
- Works with digital boards and presentations

## Technologies
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Download

Download the executable from the **Releases** section.

## Run

1. Extract the zip
2. Open the folder
3. Run `hand_tracking.exe`
